What is Smart Student Handbook About?

The Smart Student notebook is an advance note taking application that is aimed towards student. The Smart Student notebook attempts to improve, speeds up, and encourage collaboration between students. The Smart Student application will allow users to create notebooks. Notebooks can be stored in different folders. Notebooks consist of different sections and sections can then be split up into pages allowing for organized note taking. Notes are written in markup, thus allowing for multiple customization options for users such as text formatting, adding bullet points, tables, images and much more. Once a notebook is opened, a Smart assist toolbar will popup on the side , giving notebook recommendations on similar notebooks. The user will then have the ability to merge, copy or clone these notebook to their notebook. Afterwards users will be able to rate either the entire notebook or snippets of the notebook. The application will work on a subscription based service where users will be paid according to amount of other users using their notebook.

Features

Real-time Collaborate

The Smart Student Handbook allows users to collaborate in real-time with one other by adding their friends as a collaborator on their notes which will then share the note with their friends and allow them to work on the notebook at the same time.

Smart Assist

The Smart Student Handbook recommends relevant notes and note snippets that users can then add to their notes based off the notebook and note that the user us currently working on.

Chat rooms

The Smart Student Handbook has a dedicated chat room created for every notebook so that users can easily communicate and collaborate seamlessly within the same application.

Clone

The Smart Student Handbook allows users to search and explore notebooks and notes created by other users to help them get started to create their own notes by cloning an existing notebook and then having the ability to add their own content to that note.
